


# ffmpeg

## 结构
- file (muxer)
- stream (-codec)
- packet (encoder)
- frame (-filter)

## 音视频异同
- duration
- bitrate
- encoding
  - depth: sample_fmt/pix_fmt
- rate: samplerate/framerate
- layout: channels/size+aspect

## 命令行
- 时间
  - -ss, -t, -itsoffset
  - -sseof, -to
- 通用
  - -frames:v/a
  - -codec, -c:a/v
  - -filter, -af/vf
    - -filter_complex/lavfi
    - -bsf:a/v
  - -q:a/v
- 空间
  - 通用
    - rate
  - 音频
    - -sample_fmt
    - -ac
    - -aq
  - 视频
    - -pix_fmt
    - size, aspect
