# ffmpeg

## 结构
- file (muxer)
- stream (-codec)
- packet (encoder)
- frame (-filter)

## 音视频异同
- duration
- bitrate
- encoding
  - depth: sample_fmt/pix_fmt
- rate: samplerate/framerate
- layout: channels/size+aspect
