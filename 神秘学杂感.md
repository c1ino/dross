## 六元素
气木土金水火
- 气/风、木/有机、土/无机、金/金属、水/冷、火/热
- 二元
  - 风水（地势、地利）
  - 风雨（天气、露天）
  - 风土（环境、风俗）
  - 水土（饮食、食物）
  - 土木（建筑、工程）
- 三元
  - 气木土 = 雷电 = 大气+树木+土地
  - 木土金 = 浇筑（铸造、水泥）
  - 土金水 = 采矿（淘金、洗矿）
  - 金水火 = 发电 = 涡轮+蒸汽+烧水
  - 水火气 = 蒸汽（蒸煮、蒸馏）
  - 火气木 = 燃烧 = 火三角
  - 气土水 = 三态 = 气液固
  - 木金火 = 冶炼
## 五行相生
- 木生火（燃料）
- 火生土（灰烬）
- 土生金（矿石）
- 金生水（冷凝）
- 水生木（润物）
